# - author: Who wrote the post
# - categories: List of categories for filtering
# - image: Thumbnail image for listings (put in same folder or images/)
# - draft: true  # Uncomment to hide from listing

## How to Create a New Post

1. **Create a new file** in the `posts/` folder
2. **Name it** using the format: `YYYY-MM-DD-short-title.qmd`
3. **Copy the YAML header** from this file and update the fields
4. **Write your content** using Markdown

## Markdown Formatting Examples

Here are some common formatting options:

### Text Formatting

- **Bold text** using `**text**`
- *Italic text* using `*text*`
- [Links](https://quarto.org) using `[text](url)`

### Images

To add an image, place it in your `images/` folder and reference it:

```markdown
![Description](images/your-image.jpg)
```

### Code

For inline code use backticks: `like this`

For code blocks:

```python
# Example Python code
def hello():
    print("Hello, world!")
```
