# Images Folder

Place your website images in this folder.

## Suggested Images to Add:

- `hero-image.jpg` - Main image for homepage (recommended size: 800x600px or larger)
- `project1-image.jpg` - Images for research projects
- `project2-image.jpg`
- `project3-image.jpg`
- `favicon.ico` - Small icon that appears in browser tab (16x16 or 32x32px)
- `logo.png` - If you have a lab/personal logo

## Image Tips:

1. **File formats**: Use .jpg for photos, .png for graphics with transparency
2. **File size**: Optimize images for web (aim for under 500KB per image)
3. **Dimensions**: 
   - Hero images: 1200-1600px wide
   - Thumbnails: 300-400px wide
   - Profile photos: 300x300px (square works best)
4. **Alt text**: Always add descriptive alt text in your .qmd files for accessibility

## Free Image Resources:

- [Unsplash](https://unsplash.com) - Free high-quality photos
- [Pexels](https://pexels.com) - Free stock photos
- [Biorender](https://biorender.com) - Scientific illustrations (subscription)

## Optimization Tools:

- [TinyPNG](https://tinypng.com) - Compress PNG/JPEG files
- [Squoosh](https://squoosh.app) - Google's image compression tool
