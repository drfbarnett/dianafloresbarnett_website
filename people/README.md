# People Folder

Place team member profile photos in this folder.

## Naming Convention:

Use consistent, descriptive filenames:
- `pi-photo.jpg` - Principal investigator
- `firstname-lastname.jpg` - Individual team members
- `postdoc1-photo.jpg` - Generic naming if preferred

## Photo Specifications:

- **Size**: 300x300 pixels (square) works best
- **Format**: .jpg or .png
- **Style**: Professional but approachable headshots
- **Background**: Consistent backgrounds look more polished

## Tips for Good Profile Photos:

1. Use similar lighting and backgrounds across all photos
2. Face the camera directly or at a slight angle
3. Use natural, friendly expressions
4. Crop consistently (head and shoulders)
5. High resolution, but optimized for web (~100-200KB)

## Optional: Individual Profile Pages

You can create detailed profile pages for each person:

1. Create `firstname-lastname.qmd` in this folder
2. Include detailed bio, publications, CV link
3. Link to it from the main people.qmd page

Example file structure:
```
people/
├── README.md
├── pi-photo.jpg
├── pi-profile.qmd       # Optional detailed page
├── student1-photo.jpg
└── student1-profile.qmd # Optional detailed page
```
